## Getting Started

- npm install
- npm run dev
- http://localhost:3000/

## Tech Stack & Architecture

- Next.js (App Router)
- React (v19)
- TypeScript
- Redux Toolkit
- TanStack React Query
- Axios
- Tailwind CSS

---

## Folder Structure

- app/ → routing and pages
- components/ → reusable components
- services/ → API calls
- store/ → Redux setup
